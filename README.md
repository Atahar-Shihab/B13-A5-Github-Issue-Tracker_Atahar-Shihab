1. What is the difference between var, let, and const?
Answer: var is the old way to create variables and can cause bugs. let is the new way for variables that can be changed later if necessary and const is for variables that will never change.

2. What is the spread operator (...)?
Answer: The spread operator is three dots (...). It unpacks elements from an array or object so we can easily copy or combine them.

3. What is the difference between map(), filter(), and forEach()?
Answer: map() changes every item in an array and gives us a new array. filter() gives us a new array with only the items that pass a test. forEach() just loops through the array to do an action, but does not make a new array.

4. What is an arrow function?
Answer: An arrow function is a shorter and cleaner way to write a normal function in JavaScript. Instead of using the word function, you use an arrow symbol =>.

For example:
function add(a, b) { return a + b; }

we can write it as:
const add = (a, b) => a + b;

5. What are template literals?
Answer: Template literals use backticks (`) instead of regular quotes. They helps us to easily put JavaScript variables directly inside a string of text using the dollar sign and curly braces.
